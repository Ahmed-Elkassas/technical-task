## Run Locally

Clone the project

```bash
  git clone https://github.com/Ahmed-Elkassas/technical-task.git
```

Go to the project directory

```bash
  cd technical-task
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
